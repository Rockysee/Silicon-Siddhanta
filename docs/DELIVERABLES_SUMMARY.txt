================================================================================
CORE FUNCTIONS IMPLEMENTED
================================================================================

1. search_channel_videos(query, max_results)
   - Search YouTube by topic/channel
   - Returns video list with ID, title, URL
   - Uses yt-dlp library

2. download_transcript(video_id, languages, output_dir)
   - Download single video transcript
   - Supports multiple languages (Hindi, English, Sanskrit)
   - Rate limiting and error handling built-in

3. batch_download_transcripts(video_ids, output_dir, languages)
   - Download multiple transcripts efficiently
   - Progress tracking and error handling
   - Returns success/failure mapping

4. clean_transcript(raw_text)
   - Remove timestamps and artifacts
   - Eliminate duplicates
   - Normalize whitespace
   - Strip URLs and HTML entities

5. translate_transcript(hindi_text, target_language, chunk_size)
   - Hindi to English translation
   - Google Translate backend
   - Automatic chunking for long texts
   - Rate limiting and error handling

6. extract_astrological_terms(text)
   - Find ~150 Jyotish-specific terms
   - Count occurrences
   - Return frequency dictionary
   - Covers planets, houses, nakshatras, yogas, dashas

7. generate_summary_report(transcripts_dir)
   - Comprehensive analysis of all transcripts
   - File statistics and character counts
   - Term frequency analysis
   - Top 20 terms listing

Plus utility functions: validate_video_id(), save_json_metadata()
