**Step 2: Transcribe Hindi Audio to Text**
```bash
# Use OpenAI Whisper for Hindi transcription
whisper audio_file.mp3 --language Hindi --model large-v3
# This produces .txt, .srt, and .vtt files with Hindi text
```

**Step 3: Translate Hindi to English**
```python
# Use Google Translate API or similar
from googletrans import Translator
translator = Translator()

with open("transcript.txt", "r") as f:
    hindi_text = f.read()

# Translate in chunks (API limits)
chunks = [hindi_text[i:i+5000] for i in range(0, len(hindi_text), 5000)]
english_text = ""
for chunk in chunks:
    result = translator.translate(chunk, src='hi', dest='en')
    english_text += result.text + "\n"

with open("transcript_english.txt", "w") as f:
    f.write(english_text)
```
